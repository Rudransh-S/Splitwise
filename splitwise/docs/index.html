SplitWise

Split bills and track shared expenses with friends.

A full-stack expense-splitting app — create groups, add expenses four different ways, see who owes whom, and settle up.

View on GitHub
Node.js + Express SQLite (node:sqlite) Vanilla HTML/CSS/JS Installable PWA

Features

Dashboard
Balance summary, your groups, recent expenses and activity at a glance.
Groups
Create groups with an emoji logo, add members, and track balances per group.
Expenses
Four split modes — equally, exact amounts, percentage, or shares.
Balances & Friends
Net balance with everyone you share a group with, aggregated across groups.
Settle up
Record a payment with an editable amount — partial payments included.
Statistics
Spend over time and by group, with real charts.
